HTML

Hypertext Markup Language(HTML) is the stadard markup lanuage for creating web pages ans web applications. web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, image and other objects, such as interactive forms, may be embadded into the rendered page. it provides a means to create structured documents by denoting structral semantics for text such as headings, paragraphs, lists, link, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
